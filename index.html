How to Play — TallTale

What is a Tall Tale?

A tall tale is an exaggerated, often fictional account of events. In TallTale, students are encouraged to keep talking — adding details, inventing things, building on ideas — just to keep the words flowing. The goal is fluency, not accuracy. The more they talk (and the taller the tale), the more points they earn.

Modes

  • Picture Mode: Upload images. Students describe what they see — and invent what they don't.
  • Prompt Mode: Enter your own text prompts. Students speak about the topic.
  • Prompt Library: Choose an age group and level. The game auto-selects prompts, which you can preview, swap, or browse before starting.

Setup

  • Choose the number of teams (2–10) and rounds (2–10).
  • You need rounds × teams prompts or images total.
  • Optionally add up to 5 target words per prompt — students earn bonus points for each one they use.
  • In Picture Mode, you can collect images from students beforehand for personalisation, or add your own.

Timer & "That's All!"

  • Each round has a timer (default: 30s Round 1, +15s per round).
  • Press Start Timer when the student begins speaking.
  • If the student stops talking for 5 full seconds, click That's All!
  • If the timer runs out without That's All being clicked, the team earns full points.
⏱ The 5-second rule is the teacher's call. A brief pause to think is fine — 5 seconds of silence means they've stopped.

Scoring

  • Full points — timer runs out, That's All never clicked.
  • Half points — That's All clicked after the halfway point.
  • No points — That's All clicked before halfway.
  • Base points increase each round: Round 1 = 50 pts, Round 2 = 100 pts, and so on.
  • Target word bonus: +10% of base score for each target word used. Maximum +50% if all 5 are used.
📊 Example: Round 2 (100 pts base), full time, 3 target words used → 100 + 30 = 130 pts

Target Words

  • Target words appear as clickable cards during the student's turn.
  • As the student uses each word, the teacher clicks it — it gets crossed off.
  • Each crossed-off word adds +10% to the score.

Wildcards 🃏

  • Some turns come with a wildcard, shown before the timer starts.
  • Double: ×2 base score for the round.
  • Bonus: +25% base score.
  • Half: ×0.5 base score — a risk round!
  • Target word bonuses apply on top of the wildcard modifier.
  • Wildcards can be turned on/off in setup, with adjustable frequency.

Mystery Pick 🎤

  • When Mystery Pick is on, each team sees 5 microphones on screen and picks one to reveal their prompt or picture.
  • Some microphones may hide a wildcard — the team sees it when they pick, before the timer starts.
  • After each team picks, the microphones are shuffled fresh — no team is left with fewer options.
  • If only a few prompts remain, all 5 microphones lead to one of those remaining — the pick still feels exciting!

Team Format

  • One student speaks for the whole round, or team members take turns within the round.
  • Different students can take different rounds for variety.
  • The teacher monitors and decides how teams organise themselves.

Prompt Library

  • Choose an age group and a level (1 = A2, 2 = B1, 3 = B2–C1).
  • The game auto-selects the right number of prompts for your setup.
  • On the preview screen, edit any prompt inline, swap it for a random one, or browse the full list by category.
✏️ All prompts are editable before the game starts. You can also edit them in the HTML file directly — they are clearly labelled near the top of the code.

Assist! 💡

  • When Assist! is on, follow-up scaffold questions appear below the main prompt in smaller text.
  • These are visible from the start of the turn, to help students who get stuck find more to say.
  • In Picture Mode: type up to 3 scaffold questions that apply to all pictures (e.g. "What do you see first? / What is happening? / How does it make you feel?").
  • In Prompt Mode: upload a CSV file with scaffold columns, or leave scaffolds empty for manual prompts.
  • In Prompt Library: add scaffold questions per prompt on the preview screen, or upload a CSV.
📋 CSV format: columns named prompt, target_words (optional), scaffold_1, scaffold_2, scaffold_3. First row = headers. Save as CSV before uploading.
